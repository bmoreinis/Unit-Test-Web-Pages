How To Do This Quiz

These instructions are in index.html AND in instructions.html. When you press Run, it only shows you index.html - so you have to keep track of what you are putting into index.html and where it came from. Also - these instructions have examples of all the code you need to do this test!

What to do

  1. Set up index.html

    1. Get the problem code
      1. Go to the next page you are working on (like 1sections.html)
      2. Get the code:
        1. Click inside the code
        2. Press Control+A to select all
        3. Press Control+C to copy it
    2. Paste it into index.html
      1. Click inside index.html
      2. Press Control+A to select all
      3. Press Control+V to paste it

  2. Fix Code

    1. Look at the top of the page for an HTML comment. It tells you which image the page should look like and what to do.
    2. Press Run to see what the page looks like now.
    3. Fix the code on the page. There may be indentation problems (if the instructions say) as well as code problems.
    4. Press Run when you think it is done, and see if it matches the picture of what it is supposed to look like.

  3. Paste from index.html to original

    1. When the code looks good and the image matches, press Control+A to select all of the code, then Control+C to copy it.
    2. Go back to the original page (like 1hello.html) and press Control+A to select all, then Control+V to paste the new code on top.

  4. Next Page

    Go to the next page to work on (like 2survey.html). To read these instructions again. get them from instructions.html (because you removed them from index.html) and paste them onto index.html.

Completing This Quiz

Keep pasting the HTML over index.html, reworking it, and then pasting it back where it belongs.

When done, submit the link to your work to the Web Unit Test assignment in Classroom. The link should begin with https://replit.com/@.
